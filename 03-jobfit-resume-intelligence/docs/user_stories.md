# User Stories

## Applicant

- As an applicant, I want to score job descriptions against my resume so that I can apply to roles where I have the strongest evidence.
- As an applicant, I want missing keyword suggestions so that I can choose which skills to learn next.
- As an applicant, I want a weekly report so that I can maintain application momentum.

## Career coach

- As a career coach, I want to see why a role was prioritized so that I can challenge weak assumptions.
- As a career coach, I want the student to write next actions, not just collect job links.
