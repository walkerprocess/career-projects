## Current Direction

I am a Kelley School of Business student studying Management Information Systems and Business Analytics. These projects are built around three goals:

- Use data to support operational and product decisions.
- Build practical tools that are simple enough to explain in interviews.
- Connect technical implementation with business impact.

## Tech Stack

- Python for analytics, RSS/API ingestion, and report generation
- Node.js and WebSocket for real-time communication
- Web Crypto API for browser-side encryption
- SQL-style business analysis and KPI framing
- Markdown reports for recruiter-readable project summaries

## Running The Projects

Each project has its own README with setup instructions. The Python projects use only standard-library modules, and the chatroom uses Node.js dependencies listed in its `package.json`.
